Unitree G1: Guida Operativa Completa

Panoramica completa sul funzionamento e programmazione a 360° del robot umanoide Unitree G1, l'ultimo prodotto di casa Unitree che punta a rivoluzionare l'automazione.

Modelli Disponibili

  • G1-EDU Standard: 23 Gradi di Libertà, privo di manipolatori
  • G1-EDU Advanced: 29 Gradi di Libertà, privo di manipolatori
  • G1-EDU Ultimate A: 29 DoF + mani Dex3-1 a 3 dita senza sensori
  • G1-EDU Ultimate B: 29 DoF + mani Dex3-1 a 3 dita con sensori
  • G1-EDU Ultimate C: 29 DoF + mani a 5 dita con sensori

Specifiche Tecniche

Il robot G1 è diviso in due sezioni principali: parte superiore e inferiore del corpo.

  • Braccio singolo: 5 gradi di libertà (spalla, parte superiore del braccio, gomito)
  • Gamba singola: 6 gradi di libertà (anca, gamba, ginocchio, caviglia)
  • Vita: 1 grado di libertà (articolazione lombare)

Guida All'Utilizzo

Quick Start

Procedure di avvio consigliate:

  1. Avvio regolare: Boot → Damping Mode → Lock Stand → Main Operation Control → Demo → Seated Mode → Spegnimento
  2. Avvio da sdraiato: Boot (corpo piatto) → Damping Mode → Lying and Standing → Demo → Squat Switch → Spegnimento
  3. Avvio accovacciato: Boot (Squat Mode) → Damping Mode → Squat Switch → Demo → Spegnimento

Controllo Remoto

Il G1 può essere controllato tramite controller remoto con funzioni avanzate per movimento, calibrazione e debug.

Interfacce di Comunicazione

Il robot supporta diverse interfacce per la comunicazione e il controllo:

  • Ethernet per comunicazione ad alta velocità
  • WiFi per controllo wireless
  • USB per configurazione e debug
  • SDK Unitree per programmazione avanzata

Sistemi Radar e Depth Camera

Il G1 è equipaggiato con sistemi avanzati di percezione:

  • LiDAR per mappatura e navigazione
  • Depth Camera per rilevamento oggetti
  • Sensori di visione per riconoscimento
